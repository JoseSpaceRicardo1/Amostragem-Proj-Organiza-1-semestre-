# Plano de Testes de Software

| CT-01: Tela Login | RF-13| Verificar se os links da página de login estão funcionando corretamente, redirecionando o usuário para a página de recuperação de senha e de cadastro. | 1. Visualizar a página de login.<br>2. Clicar no link "Esqueceu sua senha?" e verificar se redireciona para a página de recuperação.<br>3. | CT-02: Recuperação de senha | RF-14 | Verificar se o sistema envia corretamente a solicitação de redefinição de senha para o e-mail informado. | 1. Clicar no link “Esqueceu sua senha?”.<br>2. Informar um e-mail válido na tela de redefinição.<br>3. Clicar no botão “Enviar”. | O sistema deve confirmar o envio de uma nova senha ou link de redefinição para o e-mail cadastrado. | José Ricardo Feitosa Marcelino |
| CT-03: Cadastro de Novo Usuário | RF-06 | Verificar se o sistema permite o cadastro de um novo usuário com nome, e-mail, senha e confirmação de senha. | 1. Acessar a tela de cadastro.<br>2. Preencher os campos obrigatórios.<br>3. Clicar em "Cadastrar".  | O sistema deve validar todos os campos, criar o usuário com sucesso e redirecionar ou exibir mensagem de confirmação. | CT-04: Tela sem tarefa adicionada  | RF-02, RF-03, RF-04, RF-10  | Verificar a exibição correta da tela quando não há tarefas adicionadas. | 1. Acessar a tela “Iniciar Ciclo”.| Exibir mensagem “Nenhuma tarefa adicionada!” com ícone ilustrativo. | Mariana Rocha Silva |
| CT-05: Adicionar Tarefas| RF-1, RF-2 | Verificar se o sistema permite adicionar tarefas com nome, descrição e peso.| 1. Clicar em "Adicionar Tarefa".<br>2. Preencher os campos.<br>3. | CT-06: Adicionar Membro| RF-15 | Verificar se o sistema permite adicionar um novo membro ao grupo através do e-mail | 1. Acessar o sistema com login válido.<br>2. Clicar no botão “Adicionar”. | O sistema deve validar o e-mail e adicionar o membro com sucesso ao grupo, exibindo confirmação. |Maria Luiza Campos Pereira|
| CT-07: Consulta ao Histórico de Tarefas | RF-10, RF-12  | Verificar se o sistema permite visualizar o histórico de tarefas realizadas e não realizadas, com filtros por tarefa e responsável. | 1. | CT08: Menu de configurações|RF‑14| Verificar se o menu com opções "Perfil", "Pessoas no ciclo" e "Sair" é exibido corretamente ao clicar no ícone do usuário | 1. Acessar o sistema.<br>2. | CT08.1: Editar perfil do usuário|RF‑14| Validar se o usuário consegue editar nome, e-mail e senha| 1. Acessar o menu do usuário.<br>2. Clicar em "Perfil".<br>3. Alterar os dados.<br>4. Clicar em "Salvar".| Os dados são atualizados e salvos corretamente no sistema.| Nathalia Felix Rosa Candioto e Maria Luiza Campos Pereira|
| CT08.2: Excluir conta|RF‑14| Verificar se o botão "Excluir conta" remove o perfil do sistema| 1. Acessar a tela de edição de perfil.<br>2. | CT-09: Ciclo Ativo de Atividades | RF-03, RF-04| Verificar se o sistema distribui automaticamente as tarefas entre os membros de forma equilibrada com base no peso. | 1. Adicionar membros.<br>2. Adicionar tarefas com pesos variados.<br>3. Clicar em “Iniciar Ciclo”. | Tarefas devem ser distribuídas entre os membros considerando o peso para equilíbrio da carga. | Maria Luiza Campos Pereira |
| CT-10: Edição de Tarefa | RF-02, RF-05| Verificar se o sistema permite editar corretamente os campos de uma tarefa já cadastrada. | 1. Acessar uma tarefa existente.<br>2. Clicar em “Salvar”. | A tarefa deve ser atualizada corretamente no sistema com os novos dados informados.<br>Usuário deve receber feedback visual (ex: mensagem ou redirecionamento). |Nathalia Felix Rosa Candioto|
| CT-11: Visualização de Instruções | RF-08 | Verificar se o sistema exibe corretamente as instruções de uso para os usuários. | 1.
