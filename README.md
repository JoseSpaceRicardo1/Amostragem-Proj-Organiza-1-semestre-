# Organiza +

`Análise e Desenvolvimento de Sistema`

`Eixo 1 - Projeto: Desenvolvimento de Aplicação Web Front-End`

`1º Semestre de 2025`

É um aplicativo para gestão doméstica que tem como objetivo equilibrar a distribuição de tarefas entre os moradores, seja em famílias numerosas, residências compartilhadas ou para quem mora sozinho. A solução permite organizar, delegar e acompanhar atividades domésticas, promovendo maior eficiência e colaboração. Além disso, pode incluir funcionalidades como monitoramento de pendências e personalização conforme as necessidades dos usuários.
